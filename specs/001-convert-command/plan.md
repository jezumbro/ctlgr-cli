# Implementation Plan: Add `ctlgr convert` Command

**Branch**: `worktree-feat+convert-command` | **Date**: 2026-05-30 | **Spec**: [spec.md](spec.md)
**Input**: Feature specification from `specs/001-convert-command/spec.md`

## Summary

Add `ctlgr convert` — a dedicated subcommand that converts `.md` files to `.html` catalog
documents and removes the source files. Simultaneously remove the `prefer-html` rule and all
conversion logic from `ctlgr lint`. The conversion functions already exist in `src/lint.rs`
and are moved wholesale to a new `src/convert.rs` module.

**Performance Goals**: N/A — conversion is I/O bound; batch sizes are small
**Constraints**: No new dependencies; no new abstractions beyond what the issue requires
**Scale/Scope**: Single-binary CLI; one new source module (~100 LOC)

## Phase 1: Design (Complete)

### API Design: `src/convert.rs`

```rust
#[derive(clap::Parser)]
pub struct ConvertArgs {
    #[arg(short, long, value_name = "file")]
    pub file: Vec<String>,

    #[arg(long, value_name = "dir")]
    pub dir: Option<String>,

    #[arg(long)]
    pub dry_run: bool,
}

pub fn run(args: &ConvertArgs) -> Result<()>
pub fn convert_md_to_html(md_path: &str, dry_run: bool) -> Result<()>
pub fn md_html_path(md_path: &str) -> String
pub fn md_to_html(content: &str) -> String
pub fn md_to_html_fragment(content: &str) -> String
pub fn merge_html(existing_html: &str, fragment: &str) -> String
```

### `run()` logic

1. Collect files: `--file` list, or `--dir` glob for `*.md`, or configured catalog dir
2. For each `.md` file: call `convert_md_to_html(path, args.dry_run)`
3. Accumulate errors; exit non-zero if any failed

### `convert_md_to_html(md_path, dry_run)` logic

```
read source .md
derive html_path via md_html_path()
if html_path exists:
    read existing html
    fragment = md_to_html_fragment(source)
    merged = merge_html(existing, fragment)
    if !dry_run: write merged to html_path; remove md_path
    println!("{md_path}: merged into {html_path}")
else:
    html = md_to_html(source)
    if !dry_run: write html to html_path; remove md_path
    println!("{md_path}: converted to {html_path}")
```
